Step 5: Understanding User Links

Two types of user links:

1. Shared User Link (from Step 4)
  • Generic link that allows anyone to register
  • Users access this link and submit registration request
  • Optional: Store in "User Links" tab for backup
2. Personalized User Links (Auto-generated)
  • When you approve a registration, a personalized link is automatically generated for that user
  • Each user gets their own unique link with tracking
  • Automatically stored in "UserLinks" tab with user's name
  • Share this personalized link with the approved user
