1
Create Google Sheet

This will store all your expense data. Each expense group needs its own Sheet.

üìù Actions:

  • Go to sheets.google.com
  • Click "+ Blank" to create new spreadsheet
  • Rename it (e.g., "Sabarimala 2026 Expenses")
  • Create 4 sheets (tabs) at the bottom:
Participants Expenses Users Settlements

‚ö†Ô∏è Important: Create all 4 sheets exactly as named above. Don't add or remove sheets later.

2
Set Up Google Cloud (One-time)

This enables Google Sign-In. You only need to do this ONCE for all your expense groups.

üîë Get OAuth Client ID:

  • Go to console.cloud.google.com
  • Create a new project (e.g., "Expense Tracker")
  • Enable "Google Sheets API"
  • Go to "APIs & Services" ‚Üí "Credentials"
  • Click "Create Credentials" ‚Üí "OAuth Client ID"
  • Choose "Web application"
  • Add authorized origins:
https://yourusername.github.io http://localhost:8080
  • Copy the Client ID (looks like: xxx.apps.googleusercontent.com)

üí° Tip: Save this Client ID somewhere safe. You'll use it for all expense groups.

3
Deploy Apps Script Backend

This creates the backend API that connects your app to the Google Sheet.

‚ö° Deploy Steps:

  • In your Google Sheet: Extensions ‚Üí Apps Script
  • Delete the default code
  • Copy code from google-script-oauth.gs
  • Update these two lines:
const GOOGLE_CLIENT_ID = 'YOUR_CLIENT_ID.apps.googleusercontent.com'; const ADMIN_EMAIL = 'your.email@gmail.com';
  • Click "Deploy" ‚Üí "New deployment"
  • Type: "Web app"
  • Execute as: "Me"
  • Who has access: "Anyone"
  • Click "Deploy"
  • Copy the Web App URL

üîí Security: "Anyone" means anyone with the URL. Users still need Google login and admin approval.

4
Configure in App

Final step! Connect your Sheet and API to the expense tracker app.

üéØ Configuration:

  • Go to your app URL (or open setup.html)
  • Fill in the setup form:

Group Name: Sabarimala 2026

Description: Annual pilgrimage with friends

Sheet URL: https://docs.google.com/spreadsheets/d/...

API URL: https://script.google.com/macros/s/.../exec

  • Click "Save & Continue to App"
  • Sign in with Google
  • Start tracking expenses!
